Lifecycle Sebuah Chat

1

Ingestion

Chat user diterima oleh FastAPI Controller di backend.

2

Contextualize

Mengambil Persona & Knowledge Base (RAG) dari Database.

3

Inference

Ollama memproses prompt menggunakan Qwen 1.5B.

4

Streaming

Token dikirim satu per satu ke UI untuk respon instan.

Mengenal Intent

Intent adalah "Label Perintah". Ini menentukan fungsi mana yang akan dijalankan oleh ERP.

  • CREATE_PO Buka form Purchase Order
  • VIEW_REPORT Buka tab laporan data
  • CONFIRM_SAVE Tekan tombol Simpan otomatis
